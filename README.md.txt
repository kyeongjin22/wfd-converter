# WFD PDF→JSON 변환툴

## 사용 방법
1. 사이트 접속(배포 후)
2. PDF 파일 업로드 → [변환 시작]
3. 미리보기/문장 개수/수정
4. [JSON 다운로드] 혹은 [GitHub 업로드]
5. 업로드 시 최초 1회 본인 GitHub 아이디/저장소명 입력, 토큰 필수

## 배포 방법(Netlify)
1. 본 리포를 fork 또는 zip 다운로드 → Netlify "New site from Git" 연결
2. 사이트 배포 → 주소 자동 생성
3. 원하는 도메인(서브도메인) 연결 가능

## 기타
- PDF에서 영어 문장만 추출, 중복/짧은 문장 제거
- GitHub API 업로드 지원
